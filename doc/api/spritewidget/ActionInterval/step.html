  1. @override
void step (double dt)

Moves to the next time step in an action, dt is the delta time since the last time step in seconds. Typically this method is called from the ActionController.

Implementation 

@override
void step(double dt) {
  if (_firstTick) {
    _firstTick = false;
  } else {
    _elapsed += dt;
  }

  double t;
  if (this._duration == 0.0) {
    t = 1.0;
  } else {
    t = (_elapsed / _duration).clamp(0.0, 1.0);
  }

  if (curve == null) {
    update(t);
  } else {
    update(curve.transform(t));
  }

  if (t >= 1.0) _finished = true;
}
