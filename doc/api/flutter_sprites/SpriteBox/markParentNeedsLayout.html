void markParentNeedsLayout()

Mark this render object's layout information as dirty, and then defer to the parent.

This function should only be called from markNeedsLayout implementations of subclasses that introduce more reasons for deferring the handling of dirty layout to the parent. See markNeedsLayout for details.

Only call this if parent is not null.

Source 

@protected
void markParentNeedsLayout() {
  _needsLayout = true;
  final RenderObject parent = this.parent;
  if (!_doingThisLayoutWithCallback) {
    parent.markNeedsLayout();
  } else {
    assert(parent._debugDoingThisLayout);
  }
  assert(parent == this.parent);
}
